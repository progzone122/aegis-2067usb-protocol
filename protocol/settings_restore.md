# Settings Restore
Resets the default keyboard configuration

| Parameter | Value                                            |
|-----------|--------------------------------------------------|
| wValue    | 0x0207                                           |
| wLength   | 8                                                |
| data      | [0x07, 0xf0, 0xaa, 0x00, 0x00, 0x00, 0x00, 0x00] |
